# Exercise 1.12

Dockerfiles for projects

- <https://github.com/docker-hy/frontend-example-docker>.
- <https://github.com/docker-hy/backend-example-docker>.

To build project, you need to build two images:

```bash
./frontend.sh build
./backend.sh build
```

To run project, run command

```bash
./frontend.sh run
./backend.sh run
```
